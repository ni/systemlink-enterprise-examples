# Web Apps

Build a web app and host it within SystemLink to share a custom UI with other users. [Learn more in the user manual](https://www.ni.com/docs/en-US/bundle/systemlink-enterprise/page/hosting-a-web-application.html).

- **[Framework Examples](examples/Web%20Apps/Framework%20Examples/)** contains the source for examples written in various web UI frameworks.
- **[Dev Tools](examples/Web%20Apps/Dev%20Tools/)** contains useful tools for locally developing and testing web apps.
